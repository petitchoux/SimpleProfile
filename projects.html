Litterly

Swift, Firebase, Firestore, Google Maps, Location API

Litterly is an iOS application made to encourage communities to come together to clean up the environment by allowing users to report locations of trash and to schedule meetups. By attending a meetup, the user will be awarded points.

plattr

HTML, CSS, JavaScript, jQuery, Bootstrap, Fotorama

plattr is an interactive web application inspired by Tinder's UI to make a statement on how easy it is for society to consume media at a fast pace and its effects. Similar to Tinder, plattr invites people to swipe through a series of photos of food. The viewer is able to stop at each one to learn more about the process of making them or move on to the next one.

Messenger

PHP, SQL, JavaScript, CSS, HTML

Messenger is a chatting web application that allows users to create an account and log in, search for other users, and send and receive messages.
